## Getting Started

> Requires [Node.js](https://nodejs.org) and [pnpm](https://pnpm.io).

```sh
# Install dependencies
pnpm install

# Start development server at localhost:4321
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```
